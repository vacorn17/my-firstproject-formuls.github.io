Формулы по геометрии

Основные элементы:
 • Периметр - P
• площадь - S
 • высота - h
 • радиусы - R (описанной окружности), r (вписанной окружности)
 • стороны - a, b, c
 • диагональ - d
 • средняя линия - m
 1. Планиметрия

Треугольники:
 • Периметр: P = a + b + c
 • Полупериметр p = (a + b + c) / 2
 • Площадь: S = 1/2 ahₐ = 1/2 ab sin C = √(p(р — а)(р — b)(p — c))
 • Радиусы: R = abc / 4S, r = S / p
 • Высота, медиана, биссектриса: hₐ = 2s / a, mₐ = 1/2 √(2b² + 2c² – a²), la = (2bc cosA/2) / (b + c)
• Прямоугольный треугольник: S = ab / 2, h = ab / c, R = c / 2, r = a + b – c/2
 • Теоремы: Пифагора a² + b² = c², косинусов c² = a² + b² – 2ab * cosC, синусов a/sinA = b/sinB = c/sinC = 2R

 Четырёхугольники:
 • Прямоугольник: P = 2(a+b), S = ab, d = √(a²+b²)
 • Квадрат: P = 4a, S = a², d = a√2
 • Параллелограмм: S = ab sin(aˆb) = ah
 • Ромб: S = a² sin(aˆb) = 1/2d₁d₂
 • Трапеция: S = ((a+b)/2)h, m = (a+b)/2

 Многоугольники:
 • Правильный n-угольник: P = na, S = 1/4na2cos(π/n), r = a/2tg(π/n), R = a/2sin(π/n)

 Окружность и круг:
 • Длина: l = 2πR
 • Площадь: S = πR²
 • Дуга l: 2πRa/360°
 • Сектор S: πR²a/360°
 • Хорда: AB = 2Rsin(a/2)
 • Сила точки: OT² = OA * OB

 2. Стереометрия

Объём, площадь:
• Куб: V = a³, S = 6a², d = a√3
 • Прямоугольный параллелепипед: V = abc, S = 2(ab+bc+ac), d = √(a²+b²+c²)
 • Призма: V = Sоснh, S = 2Sосн + Pоснh, d = √(dосн²+h²)
 • Пирамида: V = 1/3Sоснh, S = Sосн+Sбок
 • Усечённая пирамида: V = h/3(S₁+S₂+√S₁S₂)
 • Цилиндр: V = πR²h, S = 2πR(h+R)
 • Конус: V = 1/3πR²h, S = πR(l+R), l = √(R²+h²)
 • Усечённый конус: V = 1/3πh(R₁²+R₂²+R₁R₂), Sбок = π(R₁+R₂)l
 • Шар: V = 4/3πR³, S = 4πR²
 • Сферический сегмент: V = (πh²(3R-h))/3, S = 2πRh

 Расстояния:
 • Между точками: AB = √((x₂-x₁)²+(y₂-y₁)²+(z₂-z₁)²)
 • Точка до плоскости: d = (|Aₓ₀+By₀+Cz₀+D)/√(A²+B²+C²)
 • Угол между векторами: ∠a˄b = (ab)/(|a||b|)
 • Угол между плоскостями: ∠A˄B˄C = (|A₁A₂+B₁B₂+C₁C₂|)/(√(A₁²+B₁²+C₁²) * √(A₂²+B₂²+C₂²))

 3. Тригонометрия

• Основные функции: sin∠a = против. катет / гипотенуза, cos∠a = прил. катет / гипотенуза, tg∠a = sin∠a / cos∠a = против. катет / прил. катет
 • Тождества: sin²∠a + cos²∠a = 1, 1 + tg²∠a = 1 / cos²∠a, sin(180°-a) = sin∠a, cos(180°-a) = -cos∠a

 В треугольнике:
 • Закон синусов: a / sin∠A = b / sin∠B = c / sin∠C = 2R
 • Закон косинусов: c² = a²+b²-2ab*cos∠C
 • Площадь через синус: S = 1/2ab*sin∠A˄B
 • Полупериметр и радиус: tg∠(A/2) = \frac{r}{p-a}

 4. Координаты и векторы

• Точки и отрезки: AB = √((x₂ – x₁)² + (y₂ – y₁)² + (z₂ – z₁)²)
 • Прямые: (x – x₀)/a = (y-y₀)/b = (z-z₀)/c
 • Плоскости: A(x-x₀) + B(y-y₀) + C(z-z₀) = 0

 Векторы:
 • Длина: |a| = √(x²+y²+z²)
 • Скалярное произведение: ab = x₁x₂ + y₁y₂ + z₁z₂
 • Угол: cos∠a˄b = ab/|a||b|
